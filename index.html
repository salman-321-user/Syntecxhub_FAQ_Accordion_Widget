React is a JavaScript library for building user interfaces, developed by Facebook. It lets you create reusable UI components and efficiently update the DOM using a virtual DOM.
Components are independent, reusable pieces of UI. They can be functional (simple functions) or class-based (ES6 classes). They accept inputs called "props" and return React elements.
JSX is a syntax extension that looks like HTML but is used with React. It makes writing components easier. Under the hood, JSX is transformed into React.createElement calls.
Props (short for "properties") are read-only inputs passed to components. They allow data to flow from parent to child. Props are immutable — a component cannot change its own props.
State is an object that holds data that may change over time. Unlike props, state is managed within the component and can be updated using setState (class) or the useState hook (functional).
The virtual DOM is a lightweight copy of the actual DOM. When state changes, React updates the virtual DOM first, then compares it with the previous version (diffing) and efficiently updates the real DOM (reconciliation).
Hooks are functions that let you use state and lifecycle features in functional components. Common hooks: useState, useEffect, useContext. They were introduced in React 16.8.
useState is a hook that lets you add state to functional components. It returns an array with two elements: the current state value and a function to update it. Example: const [count, setCount] = useState(0);
